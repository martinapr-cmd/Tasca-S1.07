# Tasca-S1.07

# Objectius
- Practicar millores de robustesa en el codi font mitjançant gestió d'errors i excepcions.
- Crear excepcions personalitzades.
- Confecció d'expressions regulars.

# Nivell 1
- Exercici 1: Un cas clàssic de situació que ens pot generar una excepció seria dividir un nombre per 0.
Fes un programa que en el que es pugui produir aquesta situació i gestiona aquest error amb una sentència try-catch.

- Exercici 2: Recorda l'exercici 1 del nivell 1 del tema 6(PHP Avançat).
Programa en PHP diverses normes de validació pels camps del formulari. Com per exemple que el camp no sigui buit o que compleixi determinades circumstàncies simples com ser un nombre en cas de ser un camp numèric.

# Nivell 2
- Exercici 1: Crea excepcions personalitzades per tractar els possibles errors que hi pugui haver en la validació del formulari de l'exercici anterior.

# Nivell 3

Valida aquesta correcció mitjançant filtres o expressions regulars.
